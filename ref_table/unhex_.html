Name

unhex()

Examples 
String hs = "FF006699";
int hi = unhex(hs);
fill(hi);
rect(30, 20, 55, 55);
Description Converts a String representation of a hexadecimal number to its equivalent integer value.
Syntax 
unhex(value)
Parameters
value String: String to convert to an integer
Returns int

binary()
