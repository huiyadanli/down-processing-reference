Name

noCursor()

Examples 
// Press the mouse to hide the cursor
void draw() 
{
  if (mousePressed == true) {
    noCursor();
  } else {
    cursor(HAND);
  }
}
Description Hides the cursor from view. Will not work when running the program in a web browser or in full screen (Present) mode.
Syntax 
noCursor()
Returns void
Related cursor()
