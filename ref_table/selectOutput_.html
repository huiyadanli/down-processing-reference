Name

selectOutput()

Examples 
void setup() {
  selectOutput("Select a file to write to:", "fileSelected");
}

void fileSelected(File selection) {
  if (selection == null) {
    println("Window was closed or the user hit cancel.");
  } else {
    println("User selected " + selection.getAbsolutePath());
  }
}
Description Opens a platform-specific file chooser dialog to select a file for output. After the selection is made, the selected File will be passed to the 'callback' function. If the dialog is closed or canceled, null will be sent to the function, so that the program is not waiting for additional input. The callback is necessary because of how threading works.
Syntax 
selectOutput(prompt, callback)
selectOutput(prompt, callback, file)
selectOutput(prompt, callback, file, callbackObject)
selectOutput(prompt, callbackMethod, file, callbackObject, parent)
selectOutput(prompt, callbackMethod, file, callbackObject, parent, sketch)
Parameters
prompt String: message to the user
callback String: name of the method to be called when the selection is made
Returns void
