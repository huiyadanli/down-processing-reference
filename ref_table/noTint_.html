Name

noTint()

Examples 
PImage img;
img = loadImage("laDefense.jpg");
tint(0, 153, 204);  // Tint blue
image(img, 0, 0);
noTint();  // Disable tint
image(img, 50, 0);
Description Removes the current fill value for displaying images and reverts to displaying images with their original hues.
Syntax 
noTint()
Returns void
