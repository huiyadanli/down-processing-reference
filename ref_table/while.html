Name

while

Examples 
int i = 0;
while (i < 80) {
  line(30, i, 80, i);
  i = i + 5;
}
Description Controls a sequence of repetitions. The while structure executes a series of statements continuously while the expression is true . The expression must be updated during the repetitions or the program will never "break out" of while .

This function can be dangerous because the code inside the while loop will not finish until the expression inside while becomes false. It will lock out all other code from running (e.g., mouse and keyboard events will not be updated). Syntax 
while (expression) {
  statements
}
Parameters
expression a valid expression
statements one or more statements
