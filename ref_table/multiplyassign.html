Name

*= (multiply assign)

Examples 
int a = 5;
int b = 2;
a *= b;  // Sets 'a' to 10
Description Combines multiplication with assignment. The expression a *= b is equivalent to a = a * b .
Syntax 
value1 *= value2
Parameters
value1 int or float
value2 any numerical value the same datatype as value1
Related = (assign)
* (multiply)
