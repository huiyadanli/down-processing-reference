Name

! (logical NOT)

Examples 
boolean a = false;
if (!a) {
  rect(30, 20, 50, 50);
}
if (a) {
  line(20, 10, 90, 80);
  line(20, 80, 90, 10);
}

boolean a = true;
if (!a) {
  rect(30, 20, 50, 50);
}
if (a) {
  line(20, 10, 90, 80);
  line(20, 80, 90, 10);
}
Description Inverts the Boolean value of an expression. Returns true if the expression is false and returns false if the expression is true . If the expression (a>b) evaluates to true, then !(a>b) evaluates to false.
Syntax 
!expression
Parameters
expression any valid expression
Related || (logical OR)
&& (logical AND)
if
