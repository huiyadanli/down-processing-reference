Name

break

Examples 
char letter = 'B';

switch(letter) {
  case 'A': 
    println("Alpha");  // Does not execute
    break;
  case 'B': 
    println("Bravo");  // Prints "Bravo"
    break;
  default:
    println("Zulu");   // Does not execute
    break;
}
Description Ends the execution of a structure such as switch , for , or while and jumps to the next statement after.
