Name

ceil()

Examples 
float x = 8.22;  
int a = ceil(x);  // Sets 'a' to 9
Description Calculates the closest int value that is greater than or equal to the value of the parameter. For example, ceil(9.03) returns the value 10.
Syntax 
ceil(n)
Parameters
n float: number to round up
Returns int
Related floor()
round()
