Name

continue

Examples 
for (int i = 0; i < 100; i += 10) {
  if (i == 70) {  // If 'i' is 70,
    continue;     // skip to the next iteration,
  }               // therefore not drawing the line.
  line(i, 0, i, height);
}
Description When run inside of a for or while , it skips the remainder of the block and starts the next iteration.
Syntax 
continue
Related for
while
