Name

saturation()

Examples 
noStroke();
colorMode(HSB, 255);
color c = color(0, 126, 255);
fill(c);
rect(15, 20, 35, 60);
float value = saturation(c);  // Sets 'value' to 126
fill(value);
rect(50, 20, 35, 60);
Description Extracts the saturation value from a color.
Syntax 
saturation(rgb)
Parameters
rgb int: any value of the color datatype
Returns float

green()
blue()
