Name

<= (less than or equal to)

Examples 
int a = 22;
int b = 23;
if (a <= b) {
  println("variable a is less or equal to variable b ");
}
Description Tests if the value on the left is less than the value on the right or if the values are equivalent.
Syntax 
value1 <= value2
Parameters
value1 int, float, char, or byte
value2 int, float, char, or byte
Related > (greater than)
< (less than)
>= (greater than or equal to)
== (equality)
!= (inequality)
